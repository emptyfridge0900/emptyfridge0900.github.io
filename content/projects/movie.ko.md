# 기능

- OMDB에서 영화를 검색하고 결과를 표시한다. 영화만 대상으로 한다.
- 검색 결과에서 영화를 nomination list에 추가한다.
- 이미 nomination한 영화 목록을 확인한다.
- nomination list에서 영화를 제거한다.
- 사용자가 페이지를 떠나도 nomination list를 저장한다.
- 공유 가능한 링크를 생성한다.
